# QR Code Business Card & Intake Workflow

## Overview
This repository documents a practical case study focused on designing a QR code business card and a simple digital intake workflow for a small business.

The project demonstrates how accessible tools can be used to streamline information collection and organization.

## Problem
The business relied on manual communication to collect client information, making it difficult to capture and organize details consistently.

## Goal
Create a clean, mobile-friendly solution that allows clients to quickly submit information through a QR code linked to a digital form.

## Tools Used
- Canva  
- QR Code Generator  
- Google Forms  
- Google Sheets  

## Process
1. Designed a clean business card layout using Canva  
2. Generated and embedded a QR code into the card design  
3. Created a digital intake form using Google Forms  
4. Connected form responses to Google Sheets  
5. Tested the workflow for usability and accessibility  

## Outcome
The final solution allows clients to scan a QR code, submit information digitally, and have responses automatically organized in a spreadsheet, reducing manual steps.

## What I Learned
This project reinforced the value of simplicity in small business workflows and highlighted the importance of usability and testing.

## Future Improvements

- Conditional form logic  
- Integration with lightweight CRM tools  

## Notes
